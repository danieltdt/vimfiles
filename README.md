# My vim files

## Requirements

* [Vundle](https://github.com/gmarik/Vundle.vim) requirements

## Install

1. `git clone http://github.com/danieltdt/vimfiles ~/.vim`
2. `git submodule update --init --recursive`
3. `echo "source ~/.vim/vimrc" > ~/.vimrc`
4. `vim +PluginInstall`
